# Market-Pulse
**Stand: 21. Juni 2026**

Ein elegantes, performantes Echtzeit-Dashboard für weltweite Tech-Aktien im modernen Dark-Design. Das Projekt fokussiert sich auf eine saubere, strukturierte Darstellung von Live-Marktdaten im gläsernen Look (Glassmorphism) mit einer intelligenten und ausfallsicheren Simulation.

## Features
* **Modernes UI:** Tiefdunkler Hintergrund mit gläsernen Elementen (Glassmorphism) und dynamischen Neon-Trendfarben.
* **Intelligentes Raster:** Vollkommen responsives Grid-Layout, das sich automatisch an Smartphones, Tablets und große Monitore anpasst.
* **Saubere Struktur:** Leicht erweiterbares HTML- und CSS-Gerüst ohne unnötigen Ballast.
* **Intelligente Suche:** Benutzer können flexibel entweder den echten Firmennamen (z. B. "apple", "tesla") oder das offizielle Börsenkürzel (z. B. "AAPL", "TSLA") eingeben. Das System ordnet die Eingaben automatisch korrekt zu.
* **Live-Fluktuation im Sekundentakt:** Die Kurse verändern sich alle 3 Sekunden völlig unabhängig von externen Server-Sperren direkt live vor den Augen des Nutzers.
* **Dynamische Farbanpassung:** Automatische Grün- oder Rotfärbung der Kacheln via JavaScript, je nachdem, ob der aktuelle simulierte Kurs im Vergleich zum Basispreis im Plus oder Minus steht.

## Technologien
